<h1 align="center">Ksh's Korner</h1>
<h3 align="center">Personal website</h3>

## Dependencies

- git
- zola

## Build

```bash
git clone https://github.com/ksharizard/ksharizard.github.io web
cd web
git submodule update --init --recursive
zola serve
```
